# PRD 模板

本模板定义 `analyze` 命令输出的 PRD 文档结构。每个章节都标注了信息来源要求。

---

## PRD 结构（10 章）

```markdown
# [产品名称] — 产品分析 PRD

> 生成日期：[日期]
> 数据来源：[官网 URL] + 联网搜索
> 研究完成度：[X/8]

---
```

## 填写原则

1. **每条信息标注来源 URL**
2. **表格中空白项填"未找到公开信息"**，不编造内容
3. **推测性结论加 `[推测]` 前缀** 并说明推测依据
4. **数据来源汇总表不可省略**，这是可信度的基础
5. **信息缺口部分诚实列出**，宁可留白也不编造
